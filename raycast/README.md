# Raycast

## Script Command (recommended)

1. Open Raycast → **Extensions** → create or open a **Script Command** folder.
2. Copy `coomer.sh` into that folder.
3. Edit `COOMER_BIN` inside `coomer.sh` to the path of your release binary or `Coomer.app/Contents/MacOS/coomer`.
4. Reload Script Commands in Raycast.

Running the script starts a session; run again to kill the previous instance (see app pid logic), then start fresh after the first exited.
